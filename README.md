# todo-nextjs

Next.js で作ったシンプルな TODO リストアプリです。

**デモ:** https://todo-nextjs-phi-azure.vercel.app

## 機能

- タスクの追加・編集・削除
- 完了 / 未完了の切り替え
- 優先度設定（高・中・低）
- フィルター表示（未完了 / すべて / 完了済み）
- 完了済みタスクの一括削除
- データは `localStorage` に保存（ページリロード後も維持）

## 使い方

### タスクを追加する

1. 画面上部の入力欄にタスク名を入力
2. 優先度ボタン（`! 高` / `- 中` / `v 低`）で優先度を選択
3. **追加** ボタンまたは `Enter` キーで追加

### タスクを完了にする

タスク左端の丸ボタンをクリックすると完了 / 未完了を切り替えられます。

### タスクを編集する

タスクにカーソルを合わせると右側に編集ボタン（鉛筆アイコン）が表示されます。クリックするとタイトルと優先度を変更できます。

- `Enter` キー or **保存** ボタン: 変更を保存
- `Escape` キー or **キャンセル** ボタン: 変更を破棄

### タスクを削除する

タスクにカーソルを合わせると右側に削除ボタン（ゴミ箱アイコン）が表示されます。

### フィルターで絞り込む

画面中央のタブで表示するタスクを切り替えます。

| タブ | 表示内容 |
|------|----------|
| 未完了 | 未完了のタスクのみ（デフォルト） |
| すべて | すべてのタスク |
| 完了 | 完了済みのタスクのみ |

### 完了済みを一括削除する

完了済みタスクが存在する場合、画面下部に「完了済み (N件) を削除」ボタンが表示されます。

## 開発

### セットアップ

```bash
git clone https://github.com/threver/todo-nextjs.git
cd todo-nextjs
npm install
```

### ローカルで起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

### ビルド

```bash
npm run build
npm start
```
